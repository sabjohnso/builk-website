The guide covers wile's features in depth.

## Topics

- **Macros** — hygienic macros with `syntax-rules` and `define-syntax`
- **Numeric Tower** — fixnums, flonums, rationals, complex numbers, bignums
- **Libraries** — the R7RS library system and bundled SRFIs
- **Compilation** — ahead-of-time compilation and FASL caching
- **Embedding** — using wile from OCaml or C
- **Developer Tools** — LSP, DAP, and the runtime profiler
